# XSS Vulnerabilities Lab

Welcome to the XSS Vulnerabilities Lab. This lab is designed to help you understand and identify different types of Cross-Site Scripting (XSS) vulnerabilities, including Reflected XSS, Stored XSS, Blind XSS, and DOM-based XSS.

## Introduction

XSS vulnerabilities allow attackers to inject malicious scripts into web pages viewed by other users. This lab provides examples and explanations of the four main types of XSS attacks.
